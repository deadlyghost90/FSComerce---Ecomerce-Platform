## About FScommerce

FScommerce is the core engine powering Fatahshaheen OS. It is a high-performance, modular commerce solution designed for speed, reliability, and total control. We have taken the foundational commerce primitives and customized them to build an unbeatable e-commerce experience.

## Getting Started

1. **Clone the repo**
2. **Install dependencies**
3. **Run the server**
   
Visit the [Quickstart Guide](https://docs.medusajs.com/learn/installation) to set up your server.

## Built for Fatahshaheen OS

This engine provides the commerce modules and tools needed to build rich, reliable, and performant stores without reinventing core commerce logic.
